References

I have found the following to be extremely valuable in my quest to learn Cocoa and in developing FinkCommander:

  • The book Cocoa Programming for Mac OS X by Aaron Hillegass. This is a superlative introduction to Cocoa.

  • Mike Beam's Cocoa tutorials, especially the tutorials on toolbars and customizing table columns, which I borrowed from heavily in writing FinkCommander.

  • The Cocoa Dev Central web site, another great source of tutorials.

  • The open source code and programming tips for Cocoa Browser by Hoshi Takanori and Max Horn. FinkCommander uses Cocoa Browser's "MyTextView" class to make the hand icon appear over links in the Package Inspector.
