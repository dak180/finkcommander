Create binary .dmg:
	Make sure new version is set in Target:Application Settings
	Compile with DEBUGGING undefined and "Deployment" selected
	Create directory with .app, VERSION_HISTORY.html, README.html, LICENSE.txt
	Run DMG Tool on directory:  file name = finkcommander-<version>-bin.dmg
	
Create source tarball
	tar -c -z FinkCommander/ -f finkcommander-<version>-src.tgz
