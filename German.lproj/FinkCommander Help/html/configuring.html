Konfiguration von FinkCommander und Fink

Das Hauptfenster für die Einstellungen von FinkCommander ist in sechs Kategorien unterteilt: Commander, Fink, Download, Verzeichnisse, Anzeige und Umgebung. Mit den Optionen in den Kategorien Fink und Download können Sie direkt die Konfigurationsdatei Fink-Verzeichnis/etc/fink.conf verändern.

Diese Hilfeseite beschreibt die Funktionen der verschiedenen Einstellungmöglichkeiten, wo diese nicht selbsterklärend sind.

Commander

Nie nach meinem Kennwort fragen

Wählen Sie diese Option falls Sie sudo so konfiguriert haben, dass Sie nie nach Ihrem Passwort gefragt werden. Die Anleitung, um dies zu tun, finden Sie in einem Eintrag des Fink FAQ.

Fink

Pakete aus Unstable verwenden/Crypto-Pakete aus Unstable verwenden

Mit diesen zwei Optionen können Sie schnell zwischen Auslassen und Miteinbeziehen der Pakete aus Unstable wechseln. Die Pakettabelle wird, falls die Einstellungen geändert wurden, nach einem Klick auf OK oder Anwenden automatisch aktualisiert.

Das Build-Verzeichnis nach der Installation behalten

Aus der Fink man page: Die Grundeinstellung ist nein. Gewöhnlich entfernt Fink nach dem erfolgreichen Erstellen eines Paketes das Verzeichnis mit dem kompilierten Sourcecode. Falls sie diesen Ordner behalten möchten setzen Sie diese Option. Gewöhnlicherweise haben nur Entwickler von Paketen Verwendung für diese Option.

Das Root-Verzeichnis nach der Installation behalten

Aus der Fink man page: Ähnlich der Option "Build-Verzeichnis behalten". Diese Option belässt das Verzeichis, aus welchem die .deb Datei erstellt wird, auf Ihrem System. Für Paket Entwickler gedacht.

Download

HTTP Proxy/FTP Proxy verwenden, FTP im Passiv-Modus benutzen

Sezten Sie diese Felder auf die entsprechenden Werte, falls sich Ihr System hinter einer Firewall befindet.

Download Methode

MacOS X wird mit curl ausgeliefert, daher ist dies die Grundeinstellung. Befor Sie zu wget oder axel wechseln müssen Sie die entsprechende Software installieren - natürlich kein Problem mit fink.

Verzeichnisse

Verzeichnis für Fink

Ändern Sie diese Einstellung nur, falls FinkCommander den Pfad zu Ihrer Fink Installation bei Programmstart nicht finden konnte.

Anzeige

Puffergrösse auf ____ Zeilen begrenzen.

Verwenden Sie diese Einstellungsmöglichkeit um bei der Installation grosser Pakete nicht unnötig viel Speicher zu reservieren. Grundeinstellung in Terminal.app ist 10000.

Ans Ende springen wenn neue Ausgabe empfangen wird

Ist diese Option angewählt, so zeigt die Pakettabelle nach einer Sortierung automatisch wieder auf das zuvor ausgewählte Paket. Analog zu Mail.app.

Immer fink verwenden, um die Daten zu aktualisieren

Bewirkt, dass FinkCommander nach jedem Kommando, welches eine Veränderung des Zustandes Ihrer Pakete bewirken kann, die Pakettabelle vollständig neu einliest. Mehr Details.

Umgebung

Hier können Sie zusätzliche Umgebungsvariablen festlegen, die von fink und apt-get verwendet werden. Die nötigen Grundeinstellungen werden bereits für Sie vorgenommen. Falls Sie nach Abänderung der Tabelle an Funktionalität einbüssen, kann mit der Option Wiederherstellen der Grundzustand neu aufgesetzt werden.
