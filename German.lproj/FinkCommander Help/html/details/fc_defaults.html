Die Option "Bei Fragen immer Vorschlag von Fink als Antwort verwenden"

Die Option Bei Fragen immer Vorschlag von Fink als Antwort verwenden hat die gleiche Wirkung wie wenn fink -y auf der Kommandozeile ausgeführt wird. Offenbar funktioniert dies aber nicht, wenn fink ein Paket zu installieren versucht, bei dem neue User dem System hinzugefügt werden. Ein Beispiel dafür ist mysql.
Falls FinkCommander bei einer Eingabeaufforderung hängenbleibt, wählen Sie den Menüpunkt Werkzeuge:Eingabe an Fink senden... um eine Antwort einzugeben.
