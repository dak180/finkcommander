Aus dem fink FAQ (Hinweis: dies ist eine Übersetzung im Rahmen der FinkCommander Hilfe; falls Sie den Original-Text in Englisch lesen wollen, folgen Sie bitte dem Link):

Q3.9: Ich habe es satt, immer wieder mein Passwort in sudo einzutippen. Gibt es eine Möglichkeit, dies zu umgehen?

A: Wenn Sie nicht paranoid sind können Sie sudo so konfigurieren, dass Sie nie nach Ihrem Passwort gefragt werden. Editieren Sie dazu /etc/sudoers während Sie als root eingeloggt sind und fügen Sie eine Zeile hinzu, die etwa so aussieht:

username ALL = NOPASSWD: ALL

Natürlich müssen Sie username durch Ihren aktuellen Benutzernamen ersetzen. Diese Zeile erlaubt Ihnen, jegliches Kommando via sudo auszuführen, ohne dass Sie Ihr Passwort eingeben müssen.
