Binary Packages

FinkCommander uses the apt-cache program to identify packages that are available in binary format. The apt-cache generated list includes packages that the Fink project developers have made available in binary form and source packages that you have at some point installed on your system. When a package belongs the second group but not the first, selecting Binary:Install (i.e. "apt-get install") will generate an error message.
If this happens, and you want to make sure you have the latest version of the package, or you have removed the package and want to reinstall, select Source:Install.
