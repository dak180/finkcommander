Overview

FinkCommander provides a GUI frontend to the Fink package manager. You will need to install Fink itself in order for FinkCommander to be of any use.

On startup, FinkCommander loads fink package data into a table. To run a command, simply select the package(s), if any, to which the command should apply and select the appropriate command from the "Source" or "Binary" menu. For information on the Fink commands, try the on-line Fink User's Guide

A limited amount of customization is available via the FinkCommander:Preferences menu. More is on the way.

Please note that this software is still in the early stages of development. It could break your system or even present a security risk.

If you're brave enough to try FinkCommander, the developer would very much appreciate your comments.

If you encounter bugs, it would be very helpful to have them reported at the SourceForge Bug Tracker for FinkCommander.

Unfortunately, I will not be able to provide direct support for FinkCommander. Software development is my hobby. I have a pretty demanding "day job," so responding directly to every request for help is just not in the cards.

Note on Security

FinkCommander does not yet use Apple's Security Framework to authorize commands requiring administrator privileges. Instead, it asks you to enter your password in a text field and then passes it along to the sudo command. This means that you are giving your password to FinkCommander. If I were in your shoes, I would think twice about this. If FinkCommander were evil software, it could use the password do real mischief to your system.

The one protection you have against misuse is that FinkCommander is open source software. I would encourage you to obtain and look at the source code to see what the program is doing with your password.
