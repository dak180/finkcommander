Acknowledgements

The FinkCommander development team wishes to express its appreciation to:

  • Max Horn and David Morrison for carrying on so capably as project administrators after Christoph left.

  • Dave Love, for providing the code that finally enabled FinkCommander to use the Security Framework instead of sudo to authorize fink and apt-get commands.

  • The rest of the Fink development team for all the great Unix software they have made easily available to Mac users.

References

The following resources were extremely helpful in the development of FinkCommander and are highly recommended:

  • The book Cocoa Programming for Mac OS X by Aaron Hillegass. This is a superlative introduction to Cocoa.

  • Mike Beam's Cocoa tutorials, especially the tutorials on toolbars and customizing table columns, which FinkCommander borrows from heavily.

  • The Cocoa Dev Central web site, another great source of tutorials.

  • The open source code and programming tips for Cocoa Browser by Hoshi Takanori and Max Horn. FinkCommander uses Cocoa Browser's "MyTextView" class to make the hand icon appear over links in the Package Inspector.
